Agent Instructions

To build DIY Higgsfield, follow this pipeline in order:

  1. 1. Set up conda env with Python 3.12 + CUDA 12.4. Install diffusers, torch, xformers.
  2. 2. Download Stable Video Diffusion from HuggingFace. Test: generate a 2-second video from an image.
  3. 3. Collect 10 short videos (5-15s each). Extract frames at 4fps. Train LoRA for 1 hour.
  4. 4. Install InsightFace. Download buffalo_l + inswapper_128. Implement frame-by-frame face swap with temporal smoothing.
  5. 5. Install IP-Adapter FaceID. Generate 3 scenes with same character. Add background replacement via inpainting.
  6. 6. Apply LUTs for color grading. Interpolate 24fps→60fps with RIFE. Sync audio with Wav2Lip.
