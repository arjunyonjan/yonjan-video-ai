HyperFrames

Render HTML + Tailwind + GSAP into deterministic MP4 — no timeline needed.

npm i -g hyperframes v2.4.0

What It Is

HyperFrames renders a plain HTML document into a byte-identical MP4 file. The browser engine drives a virtual clock — no wall-time jitter, no GPU drift. Every run produces the same sha256 hash.

Write markup. Add data-width, data-height, data-duration. Render. That's the whole loop.

Minimal Composition

Three attributes turn HTML into video:

<div data-width="1080" data-height="1920" data-duration="15">
  <h1 data-start="0.5" data-duration="3">Hello World</h1>
  <img src="bg.jpg" data-start="0" data-duration="15">
</div>

Render: hyperframes render comp.html --out reel.mp4

Tailwind + GSAP in HyperFrames

Tailwind Styling

Use CDN Tailwind inside your composition. All utility classes work.

<link href="https://cdn.tailwindcss.com" rel="stylesheet">
<div data-width="1080" data-height="1920"
     data-duration="10"
     class="bg-darker flex items-center justify-center">
  <h1 class="text-7xl font-bold
             bg-gradient-to-r from-red-500
             to-blue-500 bg-clip-text
             text-transparent"
      data-start="0" data-duration="10">
    Reel Title
  </h1>
</div>

GSAP Animations

GSAP auto-pins to HyperFrames seek clock. No real-time drift.

<script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.12.5/gsap.min.js"></script>
<script>
gsap.to(".title", {
  scale: 1.2,
  duration: 2,
  ease: "power2.out"
})
</script>

Reel Template — 15s Vertical

1080x1920, 15-second reel with fade transitions and GSAP text animation:

<!DOCTYPE html>
<html>
<head>
<script src="https://cdn.tailwindcss.com"></script>
<script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.12.5/gsap.min.js"></script>
</head>
<body>
<div data-width="1080" data-height="1920" data-duration="15"
     class="bg-darker relative overflow-hidden">

  <div class="absolute inset-0 flex items-center justify-center"
       data-start="0" data-duration="5" data-fade="out:0.5">
    <h1 class="text-6xl font-bold text-white">Hook Line</h1>
  </div>

  <div class="absolute inset-0 flex items-center justify-center"
       data-start="4.5" data-duration="5.5" data-fade="in:0.5,out:0.5">
    <h2 class="text-4xl font-semibold text-cyan-300 px-8 text-center">
      Problem to Solution
    </h2>
  </div>

  <div class="absolute inset-0 flex items-center justify-center"
       data-start="9.5" data-duration="5.5" data-fade="in:0.5">
    <div class="text-center">
      <h3 class="text-5xl font-bold text-green-400 mb-4">CTA</h3>
      <p class="text-2xl text-gray-400">Link in bio</p>
    </div>
  </div>

</div>
<script>
gsap.from("h1", {y: 60, opacity: 0, duration: 0.8, ease: "power3.out"})
gsap.from("h2", {scale: 0.8, opacity: 0, duration: 0.6, ease: "back.out(2)"})
gsap.from("h3", {rotation: -10, opacity: 0, duration: 0.5})
</script>
</body>
</html>

Key Data Attributes

data-width / data-height

Canvas dimensions in pixels. Root element only.

data-duration

Total length (root) or track length (children) in seconds.

data-start

When this track enters the timeline (seconds).

data-fade

Cross-fade shorthand: in:0.4,out:0.4

data-trim-start

Offset into a source video asset.

data-easing

Custom cubic-bezier for fades.

Rendering Cheatsheet

# Basic render
hyperframes render comp.html --out reel.mp4

# Transparent WebM (for Premiere overlay)
hyperframes render comp.html --out overlay.webm --alpha

# Lossless master
hyperframes render comp.html --out master.mp4 --codec prores

# Parallel render (use all cores)
hyperframes render comp.html --out reel.mp4 --workers 8

# Preview in browser
hyperframes preview comp.html

Premiere + HyperFrames Pipeline

HF → Transparent WebM → Premiere → Composite + Export MP4 
# Step 1: Generate overlay in HyperFrames
hyperframes render overlay.html --out overlay.webm --alpha

# Step 2: Composite in ffmpeg (or Premiere)
ffmpeg -i raw.mp4 -i overlay.webm \
  -filter_complex "[0][1]overlay=enable='between(t,5,7)'" \
  out.mp4
